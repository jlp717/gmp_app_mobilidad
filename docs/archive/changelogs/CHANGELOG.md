# Changelog - GMP Movilidad

Todas las versiones notables de GMP Movilidad se documentan aquí.

## [1.0.0] - 2026-01-19

### Added
- Dashboard en tiempo real con KPIs de ventas, márgenes y pedidos
- Módulo de clientes con búsqueda y filtrado avanzado
- Historial completo de ventas por cliente
- Rutero del repartidor con planificación diaria/semanal
- Control de entregas con estados y firmas digitales
- Módulo de objetivos comerciales
- Análisis y predicciones de ventas
- Sistema de logging de acciones de usuario
- Tema oscuro profesional optimizado para tablets

### Security
- Autenticación JWT
- Ofuscación de código con ProGuard
- Conexiones seguras HTTPS/TLS

### Technical
- Framework: Flutter 3.0+
- Backend: Node.js con Express
- Base de datos: IBM DB2 vía ODBC
- Gestión de estado: Provider
